# 한국사 개념 비교 학습 지원 웹앱

## 프로젝트 소개
한국사 학습에서 헷갈리기 쉬운 사건, 제도, 인물, 정치 세력 등을
1:1 비교 형식으로 정리해 학습할 수 있도록 만든 웹 애플리케이션입니다.

## 개발 목적
단순 자료 열람이 아니라, 사용자가 직접 비교 주제를 찾고
복습할 수 있는 학습형 웹을 구현하는 것을 목표로 했습니다.

## 주요 기능
- 비교 주제 목록 조회
- 검색 기능
- 시대별 필터
- 비교 상세 페이지
- 사용자 입력형 복습 페이지
- localStorage 기반 최근 검색어 및 학습 템플릿 저장

## 기술 스택
- Frontend: React
- Backend: Flask
- Database: MariaDB
